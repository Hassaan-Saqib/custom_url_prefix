Custom URL Prefix

Replace the default /odoo/ path segment in all Odoo 18 backend URLs with a custom brand prefix of your choice.

Features

  • Simple configuration: One field in Settings → General Settings.
  • Immediate effect: The browser updates instantly via JS history-API.
  • Validation: Ensures safe routing with no reserved names.
  • Zero dependency: Only requires base Odoo modules.

Example

Default:

https://erp.mycompany.com/odoo/dashboard

With Custom URL Prefix:

https://erp.mycompany.com/mycompany/dashboard

Installation & Configuration

  1. Install the app from the Apps menu.
  2. Go to Settings → General Settings.
  3. Find the Custom URL Prefix section and enter your prefix.
  4. Click Save and restart your Odoo server.

Reverse Proxy Note

If you use nginx or another proxy, add a rewrite rule so direct link access works correctly:

location ~ ^/mycompany/(.*)$ {
    rewrite ^/mycompany/(.*)$ /odoo/$1 break;
    proxy_pass http://127.0.0.1:8069;
}
